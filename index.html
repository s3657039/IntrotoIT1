Project Idea : ASX Web Scraper / Stock Analysis & Research Tool

Overview

My Project Idea is a web scraper / stock analysis & research tool. Basically, to scan companies listed on the ASX and find out important details regarding the company and also being able to do research with the program allowing it to browse and scrape the web for important and relevant articles and return the most relevant and recent news on the company to make stock analysis and research a lot quicker for a simple overview. Also of course return the share price of the company and have different price analysis options available to calculate reports and statistics based on the data scraped from the program.

Motivation

My motivation is simple first of all I have an interest in the world economy and the stock market and second of all an analysis tool will save me a lot of time when researching a stock as I can run the program and get recent news articles immediately as well as the stock price and financial reports. Similar to https://simplywall.st/ however this tool is to conduct research and financial evaluation with many different calculations that are not available on other financial websites. For example I can save data and run my own calculations on it and add many different equations to calculate fields and present the findings in a nice report ready to be exported to a data type of my choice or the users choice which is very handy and will increase efficiency in company analysis.

Description

The program plan / idea is to be able to type in the code for the stock you are interested in and the program will use your input to search the web and scrape results. Providing you with the name of the company and the current stock price as well as other key statistics such as yearly-range and day range, market cap, price to earnings ratio, dividend yield. These statistics do not need to be calculated and can be scraped from the web to get live results in a neatly presented format which will save a lot of time when wanting to research a stock or even just get an update of what the price is currently at. This is the main function of the program and the overall goal of the project. However there are many other features that are lined up to be implemented after the base application / program is functional.

For example the program will have options to do many different tasks and have a menu that awill take you to different paths with different tree branches of different research and analysis tools, for example (press 1 for further information) which is an option to find out further information on the company which will do a quick web scrape and return the valid data. it can retrieve information on the company such as a little overview and description of what they are about, The CEO, number of employees, Location/ Headquarters and the date they were founded present the findings in a nice readable format.

as well as other further innovations that will come in the future down this tree line depending on research of best ways to scrape information whether ASX.com is a viable option or google finance. Based on what information is found we will have the option to conduct further research if we want to. For example, if the CEO has been found in the web scrape we can do a further web scrape and find some information about him or her using public records, articles, web searches such as Wikipedia google etc. basically this branches off into a sort of OSINT investigation tool for people and companies that will give us valuable information to learn about the business that we are looking into from an investment standpoint. This tool will be private or in beta phases until depending on how sophisticated of an analysis tool it becomes over constant improvement potentially sell it to investors who and can benefit greatly from the generated reports and extensive investigation techniques such as long-term investors or short investors looking to analyse a company.

Not only will this program be able to find information on the company, return the current price but it will also have the financial tree line option to gather financial statistics such as revenue, diluted EPS, Cash on hand, Net change in cash. Get a quick summary of Financial data and reports etc. It will scrape as much as it can and store all of the data in a database. For example Q1-2020, Q2-2020, Q1-2021. There will be a wide variety of tools here where we can manipulate this data and conduct analysis based on comparing reports to the previous quarter.

There will be options to export the data in SQL format, CSV and other formats which can be super handy and there will be many different calculations that we can find out for example a function to give a report if the company is Cash flow positive etc. or a volatility report where we decide to find the yearly, monthly, or daily percentage of how volatile the price has been by recording or scraping the different key price fluctuation points and using different equations to calculate this and provide reports that can be downloaded, exported or emailed to yourself (as well as many other calculations and reports). There are many other ideas to be incorporated but this is it so far.



Tools and Technologies

I will be coding the program in python and will need to do extensive research on the best and most effective ways to web scrape and parse HTML. I will be most likely using the requests package to connect to the web and collect valuable data and information which I will also most likely be using beautiful soup package to parse the HTML so it can be read and saved into python, so I can then manipulate the data and use it for the program. I will also be looking into selenium which is for JavaScript as I don’t know I will be able to scrape all the information purely with HTML. I will also need to install and run a web driver to cooperate with selenium and scrape data. Also open source websites with financial data like google finance etc.

I will also need to have a good understanding of programming in python and I will definitely be learning a lot along the way.
